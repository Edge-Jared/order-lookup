This directory contains SVG logo files for each supported partner in the Edge Order Lookup tool.

Place the following files here (all lowercase):

- banxa.svg
- paybis.svg
- moonpay.svg
- simplex.svg
- changenow.svg
- letsexchange.svg
- bity.svg

Logos should be official SVGs from the partner's website or a reputable crypto logo repository (e.g., simpleicons.org, cryptologos.cc).

If a logo is missing, the UI will fallback to a colored letter icon.

Example sources:
- Banxa: https://www.banxa.com/brand-assets
- Paybis: https://www.paybis.com/brand-assets/
- Moonpay: https://www.moonpay.com/press
- Simplex: https://www.simplex.com/press-kit/
- ChangeNow: https://changenow.io/brand-assets
- LetsExchange: https://letsexchange.io/press-kit
- Bity: https://bity.com/press-kit
